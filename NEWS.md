# SeuratExtra 0.1.0

## Major Changes
* Initial CRAN submission.
* Added `sc_plot()`: A unified visualization wrapper for Seurat objects.

## New Features
* `sc_plot()` supports 5 major plot types (`dim`, `feat`, `dot`, `vln`, `heat`) with a consistent API.
* Implemented professional aesthetic defaults (corner-arrow axes style for spatial/dim reductions).
